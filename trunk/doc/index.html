APC Action Application Documentation Index

There are documents written for people with different levels of experience with the ActionApps. From least to most experience, these document categories are:
  1. New to ActionApps (thinking about using ActionApps, or just starting to use them)
  2. Web Site Manager or Slice Administrator (configuring the ActionApps using the web-based interface)
  3. System Administrator (how to install the ActionApps on a UNIX server)
  4. APC-AA Developer (how to extend and contribute back to the ActionApps sourcecode)
NOTE: some of the items below link to the online manual. You can also Download the Entire Manual

1. New to ActionApps

(Documents for people thinking about using ActionApps, or just starting to use them)
  • APC-AA Homepage -- General Information
    Description:If you want to try the applications for free, start here with the description and live demonstration pages.
  • Manual: Chapter I - Overview
    Description:For nonprofit staff, written at a non-technical level. Includes Features of ActionApps, core concepts of the software, and how to plan using the applications in your web site
  • Manual: Chapter II - Author and Editors Guide
    Description:This chapter guides you through the typical author and editors tasks of using ActionApps. Here you will find step-by-step instructions that show you how to perform these tasks.

2. Web Site Manager and Slice Administrator

(Documents for people configuring the ActionApps using the web-based interface)
  • Manual: Chapter III - Webmaster's Guide
    Description: This section guides you through setting up a Action App 'slice' for your web site and other typical administration tasks. Here you will find step-by-step instructions for how to perform these tasks. One of the primary tasks of the slice administrator is to configure how your slice appears on your organization's Web site, and how ActionApps appear to staff members and volunteers.
  • FAQ for Slice Administrators
    Description:Answers on how to use advanced Slice Administrator features of the ActionApps.

3. System Administrator

(Documents for people installing the ActionApps on a UNIX server)
  • Installation Instructions
    Description: Assume Linux/UNIX-server with mysql-php-apache configured to work with each other
  • Upgrade Instructions
    Description: How to upgrade the action-applications software from version 1.6 to a recent version from CVS
  • Installation documentation
    Description: Outdated, but thorough, big picture installation documentation

4. APC-AA Developer

(Documents for programmers extending and contributing back to the ActionApps source code)
  • Open Source Management Plan
    Description: How to actual do the Open Source Development Process
  • Modular Component Specification
    Description: This document describes the ActionApps modular components. This includes a specification where actual modules can be found, and proposals for possible future directions where modules do not yet exist.
  • Cross Server Networking Specification
    Description: This document contains a feature description, an analysis and a set of options for planned development of a cross server networking capability for the APC ActionApps. This feature is scheduled for development during Fall 2001.
  • Email subscription specification
    Description: The document specifies an email subscription service for the Action Application. This feature is not currently scheduled for development.
  • Spec for email notifications and slice profile
    Description: The document contains the specification for the APC ActionApps slice profile and event email notification system. This feature is not currently scheduled for development.
