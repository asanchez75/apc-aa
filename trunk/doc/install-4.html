4. Done?

Now you should be able to use AA - point your browser to http://localhost/aa/admin/sliceadd.php3

Login with your AA admin account.

You should see the page "Slice Administration - Add Slice"

Click on Add. In the next page fill the form with required values, Insert new slice.

Now you can create categories, new users, write new articles etc.
